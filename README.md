Projekt Vlaky v C#
Možnosti vytvářet kompletní vlak - napojování různých druhů vagónů
- parní lokomotiva může mít max. 5 vagónů
- vagony jsou typu osobní (spící, klasický) a nákladní

Lze provádět rezervace pouze do osobních vagónů.
